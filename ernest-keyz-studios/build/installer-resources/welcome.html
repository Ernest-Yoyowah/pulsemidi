Welcome to PulseMIDI — a real-time MIDI visualizer, chord detector, and performance monitor built for keyboard players and producers.

This installer will set up:

  • PulseMIDI.app — standalone monitor app
  • VST3 plugin — for Ableton, Logic, FL Studio, Reaper …
  • CLAP plugin — for Bitwig, Reaper, and other CLAP hosts

All components are universal binaries — they run natively on both Intel and Apple Silicon Macs without Rosetta.
